## Jeu de données Gapminder

Gapminder est un ensemble de données démographiques et économiques couvrant de nombreux pays sur plusieurs décennies.

Variables principales :
- **pop** : population totale
- **gdpPercap** : PIB par habitant
- **lifeExp** : espérance de vie moyenne
- **year** : année
- **continent** : continent

Ce jeu de données illustre l'évolution du développement humain, les inégalités et les tendances mondiales en santé et économie.

Dans cette application, les données sont agrégées par continent et par année.

### Colonnes du DataFrame utilisé :
- `year`
- `continent`
- `pop`
- `gdpPercap`
- `lifeExp`
